# new

a new one

### Installation

From the command line:
    
    bower install new

From the bower.json file
    
    {
	    "...
	    "dependencies": {
	        "new": "0.0.0"
	    },
		...
	}
	
In your html

    <script type="text/javascript" src="bower_components/new/dist/new.js"></script>
or

    <script type="text/javascript" src="bower_components/new/dist/new.min.js"></script>
